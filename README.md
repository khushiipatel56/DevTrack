# DevTrack – Developer Progress Dashboard

DevTrack is a simple and interactive Developer Progress Dashboard built using HTML, CSS, and JavaScript. It helps users organize tasks, track their progress, and improve productivity through a clean and responsive interface.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive design
- Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📁 Project Structure

```
DevTrack/
│── frontend/
│   ├── index.html
│   ├── style.css
│   └── script.js
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the `frontend` folder.
3. Open `index.html` in your web browser.
